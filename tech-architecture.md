# 科普科技馆预约平台技术架构方案

## 1. 系统概述

本系统是一个科普科技馆预约平台，包括PC端后台管理系统和微信科普预约小程序，采用前后端分离的架构设计。

## 2. 技术选型

### 2.1 后端技术栈
- **开发语言**：Java 1.8
- **框架**：Spring Boot 2.7.x
- **微服务**：Nacos 2.0.x
- **数据库**：MySQL 8.0
- **缓存**：Redis 7.0+
- **对象存储**：MinIO
- **认证**：JWT
- **API文档**：Swagger

### 2.2 前端技术栈
- **PC端后台管理系统**：Vue 3 + Vite + Element Plus
- **微信小程序**：H5嵌入 + Vue 3 + Vite
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **UI组件库**：Element Plus (PC端)、Vant (小程序)

### 3.2 核心功能模块

#### 3.2.1 PC端后台管理系统
- **用户登录**：基于JWT的认证机制
- **系统管理**：用户管理、菜单管理、角色权限管理、平台设置、数据字典、操作日志
- **运营机构管理**：运营机构管理、运营机构用户管理、预约用户管理、预约团队管理
- **场馆和活动管理**：场馆管理、活动分类、活动管理
- **预约管理**：个人预约、团队预约、核销和到访登记
- **科普资讯管理**：资讯分类、资讯管理
- **数据分析**：流量统计、内容访问流量统计、场馆数据看板

#### 3.2.2 微信小程序
- **注册登录**：手机号绑定、微信授权
- **首页**：轮播图、目录、科普研学、科普资讯
- **预约功能**：个人预约、团队预约、活动预约报名
- **线上场馆**：场馆列表、场馆详情
- **科普资讯**：资讯分类、资讯列表、资讯详情
- **个人中心**：个人信息、预约历史、亲友信息
- **服务号通知**：预约成功提醒、预约取消通知、活动通知

## 5. API设计

### 5.1 认证相关API
- `POST /api/auth/login`：用户登录
- `POST /api/auth/logout`：用户登出
- `GET /api/auth/refresh`：刷新token

### 5.2 用户相关API
- `GET /api/user/list`：获取用户列表
- `POST /api/user/add`：添加用户
- `PUT /api/user/update`：更新用户
- `DELETE /api/user/delete`：删除用户

### 5.3 预约相关API
- `GET /api/booking/personal/list`：获取个人预约列表
- `POST /api/booking/personal/add`：添加个人预约
- `PUT /api/booking/personal/cancel`：取消个人预约
- `GET /api/booking/team/list`：获取团队预约列表
- `POST /api/booking/team/add`：添加团队预约
- `PUT /api/booking/team/cancel`：取消团队预约

### 5.4 场馆和活动相关API
- `GET /api/venue/list`：获取场馆列表
- `POST /api/venue/add`：添加场馆
- `GET /api/activity/list`：获取活动列表
- `POST /api/activity/add`：添加活动

### 5.5 资讯相关API
- `GET /api/info/list`：获取资讯列表
- `GET /api/info/detail`：获取资讯详情
- `POST /api/info/view`：记录浏览
- `POST /api/info/like`：记录点赞

## 6. 部署方案

### 6.1 开发环境
- **后端**：本地开发环境，使用IDE运行Spring Boot应用
- **前端**：本地开发环境，使用Vite开发服务器
- **数据库**：本地MySQL、Redis、MinIO

### 6.2 生产环境
- **后端**：Docker容器化部署，使用Kubernetes管理
- **前端**：静态文件部署到CDN
- **数据库**：云数据库服务
- **缓存**：云Redis服务
- **对象存储**：云MinIO服务

## 9. 技术选型理由

### 9.1 后端
- **Spring Boot**：简化开发，快速构建
- **Nacos**：服务发现与配置管理
- **MySQL 8**：稳定可靠，功能丰富
- **Redis**：高性能缓存
- **MinIO**：轻量级对象存储

### 9.2 前端
- **Vue 3**：现代化前端框架，性能优异
- **Element Plus**：成熟的PC端UI组件库，美观易用
- **Vant**：轻量级移动端UI组件库，适合小程序
- **Pinia**：Vue 3推荐的状态管理库
- **Vue Router**：官方路由库
- **Axios**：流行的HTTP客户端
